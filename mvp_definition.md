# MVP 功能定義

**日期**：2026-03-17
**版本**：v1.0

---

## 一、MVP 核心價值主張

**「不只是規劃行程，更是你的旅行法律顧問」**

Travel Helper 的核心差異化：
1. **法律資訊整合**（簽證、禁帶物品、法律禁忌）
2. **即時政策通知**（簽證變更、旅遊警示）
3. **在地文化深度**（Fun Facts 增加趣味性）

---

## 二、MVP 功能清單

### 🔴 Phase 1 - 核心功能（必須有）

#### 1. AI 行程規劃（基礎版）

**功能描述**：
- 輸入：目的地、天數、預算、旅遊風格
- 輸出：每日行程建議（景點、交通、餐飲）

**MVP 範圍**：
- [ ] 基於目的地的景點推薦
- [ ] 簡單的行程時間分配
- [ ] 預算估算（粗略）
- [ ] 支援熱門目的地（先做 10-20 個國家）

**不做（延後）**：
- ❌ 複雜多城市路線優化
- ❌ 即時價格查詢
- ❌ 預訂整合

---

#### 2. 簽證資訊查詢（核心差異化）

**功能描述**：
- 根據國籍 + 目的地，自動查詢簽證需求
- 顯示：是否需要簽證、簽證類型、所需文件、辦理時間

**MVP 範圍**：
- [ ] 簽證需求資料庫（先做台灣護照 → 熱門目的地）
- [ ] 簡易查詢介面
- [ ] 顯示基本資訊：免簽/落地簽/需簽證

**資料來源**：
- 外交部領事事務局
- 各國簽證政策 API（如有）
- 維基百科簽證政策頁面（自動化爬蟲）

**不做（延後）**：
- ❌ 自動簽證申請
- ❌ 即時政策變更通知（Phase 2）

---

#### 3. 法律禁忌 & 禁帶物品（核心差異化）

**功能描述**：
- 顯示目的地的法律禁忌
- 顯示禁帶/限制物品清單

**MVP 範圍**：
- [ ] 法律禁忌資料庫（先做 10-20 個熱門國家）
- [ ] 禁帶物品清單（通用 + 目的地特定）
- [ ] 分類顯示：藥品、食品、電子產品、現金限制等

**內容範例**：
```
🇯🇵 日本
- 禁帶：含偽麻黃鹼藥品（感冒藥）、部分處方藥
- 限制：現金超過 100 萬日圓需申報
- 法律：禁止在路上吸菸（指定吸菸區除外）

🇹🇭 泰國
- 禁帶：電子菸、電子煙具
- 法律：對王室不敬罪（嚴格執行）
- 注意：寺廟著裝規定
```

**不做（延後）**：
- ❌ 即時法律更新
- ❌ 互動式問答

---

#### 4. Fun Facts（趣味功能）

**功能描述**：
- 顯示目的地有趣的文化、歷史、趣聞
- 增加用戶黏著度和分享意願

**MVP 範圍**：
- [ ] Fun Facts 資料庫（每個國家 5-10 條）
- [ ] 在行程規劃結果中穿插顯示

**內容範例**：
```
🇯🇵 日本 Fun Facts
- 日本有超過 6 萬家便利店，平均每 2300 人就有一家
- 東京的米其林星星數量全球第一
- 日本的自動販賣機數量超過 500 萬台
```

## 三、技術需求（粗略）

### 前端
- **框架**：React Native 或 Flutter（跨平台）
- **建議**：React Native（生態成熟、人才好找）

### 後端
- **語言**：Node.js 或 Python
- **資料庫**：PostgreSQL（結構化資料）+ Redis（快取）
- **AI**：OpenAI API 或 Claude API

### 資料需求
- 簽證政策資料庫
- 法律禁忌資料庫
- 景點/餐飲資料（可接 Google Places API）
- Fun Facts 內容（需整理）

## 六、風險評估

| 風險 | 影響 | 緩解措施 |
|------|------|----------|
| 資料準確性 | 高 | 多來源驗證 + 用戶回報機制 |
| 法律責任 | 高 | 免責聲明 + 建議查詢官方管道 |
| 競爭對手模仿 | 中 | 持續優化資料深度 |
| AI 成本 | 中 | 快取 + 用量控制 |
