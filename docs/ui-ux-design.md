# UI/UX 設計規格

**版本**: 1.0
**日期**: 2026-03-17

---

## 一、設計原則

### 核心價值
- **簡潔**: 一次只做一件事
- **直覺**: 不需要說明書
- **信任**: 法律資訊要有來源

### 設計語言
- **風格**: 現代、清爽、旅遊感
- **主色**: 海洋藍 (#0EA5E9) + 日落橙 (#F97316)
- **字體**: SF Pro (iOS) / Roboto (Android)

---

## 二、色彩系統

### Primary Colors
```
Primary Blue:    #0EA5E9 (主要按鈕、強調)
Primary Orange:  #F97316 (警示、重點)
```

### Semantic Colors
```
Success:  #22C55E (簽證免簽)
Warning:  #F59E0B (需要簽證)
Error:    #EF4444 (禁止/限制)
Info:     #3B82F6 (資訊)
```

### Neutral Colors
```
Gray 50:  #F9FAFB (背景)
Gray 100: #F3F4F6 (卡片背景)
Gray 200: #E5E7EB (分隔線)
Gray 500: #6B7280 (次要文字)
Gray 900: #111827 (主要文字)
```

### 行程規劃流程

```
首頁 → 點擊「規劃行程」
  ↓
選擇目的地 → 選擇日期 → 設定預算
  ↓
選擇旅遊風格 → 設定人數
  ↓
點擊「AI 生成行程」
  ↓
載入中... (AI 生成)
  ↓
顯示行程詳情
  ↓
用戶可: 編輯、分享、查看地圖
```

### 簽證查詢流程

```
首頁 → 點擊「簽證查詢」
  ↓
確認護照國籍 (自動帶入)
  ↓
選擇目的地國家
  ↓
顯示簽證資訊
  ↓
用戶可: 收藏、分享、查看官方連結
```

## 八、響應式設計

### 手機尺寸
- 寬度: 320px - 428px
- 主要斷點: 375px (iPhone SE), 414px (iPhone 14)

### 平板尺寸
- 寬度: 768px+
- 兩欄佈局
- 卡片可並排顯示

## 九、動效設計

### 頁面轉場
- Push: 從右滑入 (iOS 風格)
- Modal: 從底部滑入
- Fade: 淡入淡出 (載入)

### 互動反饋
- 按鈕點擊: 縮放 0.95 → 1.0
- 卡片點擊: 陰影加深
- 下拉更新: 旋轉動畫

### 載入狀態
- AI 生成: 脈動動畫 + 提示文字
- 資料載入: Skeleton screen

## 十、無障礙設計

- **對比度**: 文字與背景對比度 > 4.5:1
- **點擊區域**: 最小 44px x 44px
- **字體縮放**: 支援系統字體縮放
- **VoiceOver/TalkBack**: 所有元素都有 accessibility label
- **色盲友善**: 不只依靠顏色傳達資訊 (配合圖標/文字)
